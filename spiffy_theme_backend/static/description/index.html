Why Choose Spiffy Theme

Spiffy Theme stands out as the ultimate choice for your Odoo backend, offering a seamless blend of user-friendly design and powerful customization. It transforms your Odoo experience into something cleaner, faster, and more intuitive - fully tailored to your workflow and preferences.

Spiffy comes with a 3-in-1 Theme Style, so you can switch between multiple layouts based on your preference. Whether you prefer a classic view or a more modern interface, Spiffy adapts to your workflow. With its fully responsive design, it works seamlessly across all Odoo apps and devices - desktop, tablet, or mobile.

The theme is designed for productivity. Features like Global Search, Sticky Header, Zoom In/Out, and a Quick Settings Panel make everyday tasks quicker and more efficient. You can even bookmark menus, so your most-used options are always just a click away.

Spiffy supports RTL and multi-language, allowing you to run localized interfaces across multiple regions. Its App Drawer with custom icons and Google Fonts Integration help you personalize your workspace with ease.

Best of all, Spiffy is developed by experienced professionals who understand performance. Even with all its advanced features, the theme runs fast and smooth without slowing down your backend.

With Spiffy, you don't just get a theme - you get a smarter, more elegant backend experience tailored to your needs.

Full Documentation

To help you get the most out of our theme, we've put together comprehensive documentation covering every feature in detail. We're continuously updating it with new content, Images, and videos to ensure you always have the latest guidance.


Note : A full documentation can be accessed once you purchase the theme.

1. What is Spiffy and who is it for?

Spiffy is a sleek, responsive, and customizable backend theme for Odoo 18.0 (Community & Enterprise). Developed by a team of experienced Odoo developers, it enhances UI/UX, boosts productivity, and comes with world-class support.

Note - The Spiffy Backend Theme is available separately for both Odoo Community and Odoo Enterprise editions. Please ensure you select the correct version before making your purchase.

2. Is it a single module or does it need other modules?

Yes - Spiffy is a single, standalone module. All features are included. No hidden costs. No extra dependencies.

3. How fast is support and what's included?
  • 60 days of free support is included
  • First response within 4-8 Business Hours [11:00AM to 6:00PM MON-FRI Indian Standard Time]
  • Free updates for Odoo 18.0 included
  • Full usage instructions and setup guides are available inside the Spiffy Configuration panel in your Odoo backend.
  • Compatibility with third-party modules is not guaranteed.
4. How to install the theme?
  1. Add the theme to your custom addons directory
  2. Update the App List
  3. Search and install "Spiffy"
  4. If you plan to use mobile push notifications, install this Python library:
    • pip install google-auth

6. What productivity & UI features are included?
  • Light / Dark Mode
  • Global Search
  • Bookmark Menus
  • App Drawer with CustomIcons
  • Quick Settings Panel
  • Sticky Header
  • Zoom In/Out
  • Theme Customizer Panel
  • Google Fonts Integration

Custom App Icons: You can set different icons for apps using the Icon Configuration Panel, which includes access to an icon library and image upload links.

7. Can I customize the theme appearance?

Yes, Spiffy offers full control over:

  • Sidebar layout (icon, mini, full)
  • Colors, gradients, and backgrounds
  • Appbar and font styling
  • Layout scaling

Custom App Icons: You can set different icons for apps using the Icon Configuration Panel, which includes access to an icon library and image upload links.

8. Does it support RTL and multiple languages?

Yes. Spiffy supports all Odoo default languages and is fully compatible with RTL languages like Arabic and Hebrew.

9. Can I assign the theme to specific users or groups?

By default, theme is applied globally, but it can be configured for specific users or groups.

10. Can I revert to the default theme?

Yes, just uninstall the Spiffy theme module to switch back to the original Odoo backend UI.
